# Oso Osonen Cards

Crea tus propias cartas de juego a través de esta aplicación web.

https://volkiw.github.io/cards/

## ¿Cómo funciona?

Elige un fondo y añade la información que quieres que aparezca. Abajo podrás ir guardando tus cartas.

> **NOTA:** Las cartas sólo se guardan en el almacenamiento local de tu navegador, cuando borres el historial y cookies desparecerán también, o si le das a "eliminar cartas" los datos desaparecen.

### Lista de tareas y mejoras:

- Aviso en la carga de imágenes cuando sobrepasa el peso
- Botón de "eliminar cartas" dependiente de si hay elementos o no
- Posibilidad de descargar las cartas
